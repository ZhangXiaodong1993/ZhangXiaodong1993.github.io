第二篇日记

发表于 2016-05-18   |   分类于 记录   |     |   阅读次数

烦心事

本来只是看到一篇讲给博客插入音乐的博客，想实践一下，顺便测试一下。
却突然知道了一件特别烦心的事。工作上也有点不愉快，这两天回去之后，一般就是睡觉。
从小的来，妈妈最近工作中扭到腰了，还挺严重。突然意识到我才从学校出来，父母身体却不好了，努力的速度完全赶不上他们衰老的速度啊。最近看到一句话，读得好不如长得好，长得好不如生得好。真不公平啊。
本来很早就能下班的，可是一个同事请假了，另一个同事跑得比我还快。版本要发布了，我只能留下解决可能出现的问题。还真有，就是那个跑得早的同事的问题。这还不是关键，加班也就罢了，好基友在等我下班啊，最后我就请他吃了饭啊，好不公平啊。
跟随我数年的精神依托，突然回老家就业了，从此相距甚远。不光是距离啊，生活圈子也可能没有交集了。我怎么办了，回老家考个公务员？想想能不能回去当老师？她就是去做教师。都不现实啊，迷茫了。这个对我打击最大。

我陷入困境了？

无解啊，前不久她好像情绪不稳定，突然把我跟她拉到一个微博群里，发了几条微博，她从来不发微博的，只看。这什么情况？我很纳闷。后来跟她微信聊，无意之中就聊很多，问她最近怎样啊，她就说吃、睡。说的精神状态很不好的样子。我就说我虽然也上着无聊的班，但还是会看书充实精神生活之类的，聊了很多。
之前她也有跟我说过他会辞职，当时没在意啊，谁料到这么突然，我都不知道，她就回老家了，实习老师。可能当时我以为她辞职可能有机会来到我的城市呢。谁料天意弄人。
我跟她高中同桌，大学异地，因为我的不作为，贪玩，幼稚，葬送。后来我后悔，有点纠缠，被她烦了。再也没有勇气了。后来一直默默关注，想着我也会有变厉害的时候，我会等，一直等。
现在我最想的是，我的命运差不多就这样了，等你嫁了，我也按照父母的意愿随意吧，了此余生算了。
这两天，浑浑噩噩，上班埋头敲代码，下班看点书睡觉，因为我不能思考，一思考就陷入死循环。
大雨衬托着此时的心情。
又疯狂买了4本书，一个月买4本，这个月买6本了。
